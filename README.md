# 📊 Análise de Vendas da Amazon

# Sobre o Projeto

Este projeto apresenta uma Análise Exploratória de Dados (EDA) utilizando um conjunto de dados de produtos da Amazon. O objetivo é identificar padrões e relações entre preços, avaliações, descontos e categorias de produtos, gerando insights por meio de visualizações e métricas estatísticas.

O dataset, devido ao fato de ser público, está disponível nos arquivos do projeto.
## Tecnologias Utilizadas

* Python
* Pandas
* Matplotlib
* Seaborn

## Tratamento dos Dados

Durante o desenvolvimento do projeto foram realizadas etapas de preparação dos dados, incluindo:

* Verificação de valores nulos;
* Limpeza e ajuste de colunas;
* Conversão de tipos de dados quando necessário;
* Preparação da base para análise e visualização.

## Análises Realizadas

### Distribuição de Preços

Análise da distribuição dos preços dos produtos presentes na base de dados.

### Média de Preço por Categoria

Comparação do preço médio entre as diferentes categorias de produtos.

### Relação entre Avaliação e Preço

Investigação da correlação entre avaliações dos usuários e preços dos produtos utilizando o coeficiente de correlação de Pearson.

### Relação entre Desconto e Avaliação

Análise da correlação entre percentual de desconto e avaliação dos produtos.

### Categorias com Maior Quantidade de Produtos

Identificação das categorias com maior número de produtos cadastrados.

### Categorias com Maior Valor Total Acumulado

Análise das categorias que concentram o maior valor financeiro considerando o conjunto de produtos disponível.

## Principais Conceitos Aplicados

* Análise Exploratória de Dados (EDA)
* Limpeza de Dados
* Correlação de Pearson
* Visualização de Dados
* Geração de Insights
